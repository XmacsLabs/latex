  {\tmstrong{参考公式：}}
  
  若事件A,B互斥，则
  
  $P (A + B) = P (A) + P (B)$
  
  若事件A,B相互独立，则
  
  $P (\tmop{AB}) = P (A) P (B)$
  
  若事件A在一次试验中发生的概率是$p$，则$n$次独立重复试验中事件$A$恰好发生$k$次的概率
  
  $P_n (k) = C^k_n p^k (1 - p)^{n - k} (k = 0, 1, 2, \ldots, n)$
  
  台体的体积公式
  
  $V = \frac{1}{3} \left( S_1 + \sqrt{S_1 S_2} + S_2 \right) h$
  
  其中$S_1$,
  $S_2$分别表示台体的上、下底面积，$h$表示台体的高
  
  柱体的体积公式
  
  $V = Sh$
  
  其中$S$表示柱体的底面积，h表示柱体的高
  
  锥体的体积公式
  
  $V = \frac{1}{3} Sh$
  
  其中$S$表示锥体的底面积，$h$表示锥体的高
  
  球的表面积公式
  
  $S = 4 \pi R^2$
  
  球的体积公式
  
  $V = \frac{4}{3} \pi R^3$
  
  其中$R$表示球的半径
\end{multicols}
